# Темы оформления

FontaineRTC поддерживает переключаемые цветовые темы и пользовательские схемы.

## Как пользоваться

- В шапке кнопка **🎨 <имя темы>** открывает список тем.
- Встроенные темы: **Тёмная**, **Светлая**, **Розовая**.
- **⬇ Скачать шаблон темы** — выгружает текущую тему в JSON (JSONC с комментариями:
  каждый токен снабжён описанием, что он красит).
- **⬆ Загрузить тему…** — выбрать JSON-файл; тема валидируется, добавляется в список
  и сразу применяется. Пользовательские темы хранятся в `localStorage` браузера и
  помечены крестиком для удаления.

## Формат файла темы

```jsonc
{
  "name": "My Custom Theme",
  "tokens": {
    // ── Surfaces ──
    "--ft-bg": "#0c0c0c",       // Page background behind everything
    "--ft-surface": "#15161c",  // Cards, panels, tiles
    // … остальные токены …
  }
}
```

- `name` — имя темы (отображается в списке).
- `tokens` — карта `CSS-переменная → цвет`. Допускаются HEX, `rgb()`, `rgba()`.
- Комментарии (`//`, `/* */`) разрешены — при загрузке они вырезаются.
- Отсутствующие токены берутся из значений по умолчанию; неизвестные ключи
  игнорируются (с предупреждением в toast).

## Где описаны элементы

Единый реестр токенов — [`frontend/src/theme/tokens.ts`](../frontend/src/theme/tokens.ts).
Каждый токен имеет `key`, `group`, `label`, `description`, `fallback`. Это
единственный источник правды: он используется и для применения CSS-переменных, и
для генерации шаблона, и для валидации загруженных тем.

Группы токенов: **Surfaces**, **Borders**, **Text**, **Accent**, **Status**,
**Components**. Весь UI использует только `var(--ft-…)` (см.
[`frontend/src/styles/global.css`](../frontend/src/styles/global.css)) — хардкод
цветов отсутствует, поэтому темы влияют на каждый элемент.

## Добавить новый элемент в тему

1. Добавить токен в `tokens.ts` (`key`, `group`, `label`, `description`, `fallback`).
2. Использовать `var(--ft-новый-токен)` в CSS/компоненте.
3. Прописать значение во всех встроенных темах (`frontend/src/theme/themes.ts`).

Пользовательские темы без нового токена продолжат работать — подставится `fallback`.
